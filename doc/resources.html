                            resources

                            This namespace describes strings that are valid inputs to the various native components. Some components can only take certain types of resources, i.e. n.n-spawner can only accept interactables.

                            Example

                            <a-entity n-object='res: architecture/ceiling-2w-2l'></a-entity>
