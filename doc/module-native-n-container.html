n-container

native~ n-container

A container keeps a running tally of how many objects are within its bounds, and adds and removes the states container-full and container-empty based on the current count of objects. Can fire three special events: container-full, container-empty, and container-count-changed.

Properties:
Name Type Default Description
capacity number 4

The value at which the container will fire the container-full event.
