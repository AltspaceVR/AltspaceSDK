n-spawner

n-spawner

Create an object that spawns additional non-spawning copies of itself. These copies will be physically interactive and automatically syncronized between users. Only some native resources can be spawned.

Properties:
Name Type Description
res string

The identifier for the resource you want.

Example

<a-entity n-spawner='res: interactables/basketball'></a-entity>
