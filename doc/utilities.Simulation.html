    Simulation

    altspace/utilities. Simulation

    new Simulation(configopt)

    Simluation is a helper class that lets you quickly setup an AltspaceVR app. It creates a basic scene for you and starts the render and behavior loop.

    It also automatically uses the WebGL renderer when running in a desktop browser and emulates cursor events with mouse clicks.
